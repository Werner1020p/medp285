About Glitch Arcade

This site is a creative project that merges retro game aesthetics with interactive coding. Built using HTML, CSS, JavaScript, and p5.js, it invites users to engage with visual motion and playful interaction. Inspired by 80s arcade vibes, Glitch Arcade shows how design and code can work together in fun and dynamic ways.
